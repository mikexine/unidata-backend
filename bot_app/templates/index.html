Unitaly

The aim of this project is to collect data from various universities - starting with the Italian ones - elaborate this data and make them available to anyone, in a simple JSON format.

Some data that are available may be not so useful (example) - but they are used by @UnipdBot.

You can find the docs of the available API calls here.

Related projects:

These projects are using unidata.xyz:

  • @UnipdBot Telegram Bot.

  • Unipd App: Ionic app, now available in Alpha mode for android. iOS version is not available yet - the Apple's license is expensive!

    • Want to help?

    Help is very much appreciated, both with the data collection and with the code. Most of the data are static (libraries, study rooms) but some data are scraped daily from different websites (to get the current day's menu, for example).
